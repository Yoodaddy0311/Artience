# On-Page SEO Checklist

**Optimize every element precisely: Each on-page element must be individually optimized. Missing one element can tank rankings for competitive keywords.**

## Element Optimization Requirements

| Element | Best Practice | Char Limit | Priority | Formula |
|---|---|---|---|---|
| **Title Tag** | Primary keyword + modifer + brand | 50-60 chars | Critical | `[keyword] + [modifier] \| [brand]` |
| **Meta Description** | Call searcher to action, include keyword naturally | 150-160 chars | Critical | Summarize page value + CTA |
| **H1 Heading** | One per page, include primary keyword | Unlimited | Critical | Main topic headline |
| **H2 Subheadings** | 3-5 per article, include variations | Unlimited | High | Section headlines, keyword variants |
| **H3-H6** | Support subheadings (optional) | Unlimited | Medium | Nested hierarchy for complex topics |
| **Keyword Placement** | First 100 words, naturally throughout | 1-2% density | High | 1st paragraph + headings + conclusion |
| **Internal Links** | 2-4 per article, descriptive anchor text | Unlimited | High | Link to relevant internal content |
| **Alt Text** | Describe image + keyword if natural | 125 chars max | High | Detailed, accessible description |
| **URL Slug** | Include primary keyword, hyphens only | 50 chars max | Medium | `/[keyword]-phrase-here` |

## Title Tag Formula Examples

**Formula**: `[Primary Keyword] | [Modifier/Angle] | [Brand]`

- ✅ "How to Optimize Images for Web | 2024 Guide | Artbot"
- ✅ "Best Email Marketing Tools | Comparison | TechReview"
- ✅ "Sustainable Packaging Options | Cost vs Quality | EcoBox"

## Meta Description Formula

**Formula**: Summarize value proposition + include keyword + add CTA

- ✅ "Learn the fastest way to optimize images for web. Reduce file size by 70% without losing quality. See our step-by-step guide and tools."
- ✅ "Compare top 12 email marketing tools with pricing, features, and reviews. Find the best fit for your business needs."

## Content Structure Rules

1. **H1**: Only one, includes primary keyword
2. **H2**: Support primary argument (3-5 subheadings)
3. **H3+**: Optional, break down complex H2 topics
4. **First 100 words**: Include primary keyword naturally
5. **Keyword density**: 1-2% (natural reading, not stuffed)
6. **Internal links**: 2-4 relevant links per article

## On-Page Optimization Checklist

- [ ] Title tag optimized (50-60 chars, keyword first)
- [ ] Meta description written (150-160 chars, compelling)
- [ ] H1 includes primary keyword
- [ ] H2-H6 hierarchy reflects content structure
- [ ] Primary keyword in first 100 words
- [ ] 2-4 internal links with descriptive anchor text
- [ ] All images have keyword-inclusive alt text
- [ ] URL slug follows format (keywords, hyphens only)
- [ ] No keyword stuffing (reads naturally)
- [ ] Canonical tag set if multi-version
- [ ] Robots meta tags appropriate (index, follow)
