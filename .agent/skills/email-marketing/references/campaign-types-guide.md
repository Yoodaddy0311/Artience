## Campaign Type Matrix

| Type | Purpose | Frequency | Trigger | Key Metrics |
|------|---------|-----------|---------|-------------|
| **Newsletter** | Nurture, educate, retain | Weekly / Bi-weekly | Schedule | Open rate, CTR |
| **Drip Sequence** | Automated nurture path | 3-7 emails, 2-5 day gaps | Signup / Action | Completion rate, Conversion |
| **Onboarding** | Activate new users | 5-7 emails, days 1-14 | Registration | Activation rate, Day-7 retention |
| **Product Launch** | Generate excitement, sales | 3-5 emails over 2 weeks | Calendar event | Revenue, Pre-orders |
| **Re-engagement** | Win back inactive users | 3 emails, 3-7 day gaps | 30-90 day inactivity | Reactivation rate |
| **Transactional** | Confirm actions, receipts | Per event | User action | Delivery rate |
| **Cold Outreach** | B2B lead generation | 3-5 emails, 3-5 day gaps | List / Intent signal | Reply rate, Meeting rate |

| Metric | B2B Average | B2C Average | Top 10% |
|--------|------------|------------|---------|

| **Unsubscribe** | <0.3% | <0.5% | <0.1% |
| **Bounce Rate** | <2% | <2% | <0.5% |

## Email Structure Best Practices

| Element | Specification | Impact |
|---------|-------------|--------|
| **Subject Line** | 30-50 chars, personalized, no spam triggers | +20-30% open rate |
| **Preheader** | 40-100 chars, extends subject, includes CTA | +10-15% open rate |
| **Body** | Single column, 200-500 words, scannable | +15% CTR |
| **CTA** | 1 primary (button), 1 secondary (link) max | +25% CTR |
| **Footer** | Unsubscribe, address, preferences link | Legal compliance |

## Checklist

- [ ] Campaign type matches subscriber stage
- [ ] Subject line A/B test prepared (2+ variants)
- [ ] Preheader text customized (not auto-generated)
- [ ] Single primary CTA above the fold
- [ ] Mobile preview checked (60%+ open on mobile)
- [ ] Personalization tokens tested with fallbacks
- [ ] Unsubscribe link visible and working
- [ ] Send time optimized per audience timezone
- [ ] Suppression list applied (bounces, unsubscribes)
- [ ] Post-send metrics review scheduled (24h, 72h)
