# Narrative Arc Templates

**Core: Every presentation follows a narrative arc — match structure to purpose and audience.**

## Presentation Type Matrix

| Type | Structure | Slides | Duration | Audience |
|------|-----------|--------|----------|----------|
| **Pitch Deck** | Problem → Solution → Market → Traction → Team → Ask | 10-12 | 5-15 min | Investors, Executives |
| **Status Report** | Summary → KPIs → Deep Dives → Recommendations | 12-20 | 15-30 min | Stakeholders, Management |
| **Training** | Context → Concept → Demo → Practice → Summary | 20-35 | 30-60 min | Team, New hires |
| **Sales Demo** | Pain → Solution → Demo → Proof → Pricing → Next Steps | 8-15 | 15-30 min | Prospects, Buyers |
| **Quarterly Review** | Highlights → Metrics → Analysis → Roadmap → Q&A | 15-25 | 30-45 min | Leadership, Board |
| **Lightning Talk** | Hook → One Idea → Evidence → Takeaway | 5-7 | 5 min | Peers, Conference |
| **Workshop** | Problem → Theory → Exercise → Debrief → Apply | 25-40 | 60-90 min | Participants |

## Pacing Guidelines

| Duration | Slides | Pace | Buffer for Q&A |
|----------|--------|------|----------------|
| 5 min | 5-7 | 45-60s/slide | None |
| 15 min | 10-15 | 60-90s/slide | 2-3 min |
| 30 min | 15-25 | 75-120s/slide | 5 min |
| 60 min | 25-40 | 90-150s/slide | 10 min |

## Pitch Deck Narrative Formula

1. **Hook** (1 slide): Surprising stat or bold claim
2. **Problem** (1-2 slides): Pain point with evidence
3. **Solution** (2-3 slides): Your approach, demo/visual
4. **Market** (1 slide): TAM/SAM/SOM with growth
5. **Traction** (1-2 slides): Metrics, customers, revenue
6. **Team** (1 slide): Key people, relevant experience
7. **Ask** (1 slide): Funding amount, use of funds, timeline

## Report Narrative Formula

1. **Executive Summary** (1 slide): Bottom line up front (BLUF)
2. **Scorecard** (1-2 slides): KPIs with status indicators
3. **Wins** (2-3 slides): What worked, with data
4. **Challenges** (1-2 slides): Issues, root causes
5. **Deep Dives** (3-5 slides): Detailed analysis on key topics
6. **Recommendations** (2-3 slides): Prioritized next steps
7. **Appendix**: Supporting data (reference only)

## Checklist

- [ ] Narrative arc matches presentation type
- [ ] Opening hook captures attention in 30 seconds
- [ ] One core message per slide
- [ ] Slide count within pacing guideline range
- [ ] Q&A buffer time allocated
- [ ] Closing slide has clear call-to-action
- [ ] Rehearsed within time limit ±10%
