# Slide Design Principles

**Core: One idea per slide. 8-word headline max. 60% visual, 40% text. Less is always more.**

## Slide Type Reference

| Type | Purpose | Layout | Content Density | Example Use |
|------|---------|--------|-----------------|-------------|
| **Title** | Open/close sections | Center-aligned, large text | Minimal | Presentation opener |
| **Statement** | Bold claim or insight | Full-width text, large font | 1 sentence | Key takeaway |
| **Content** | Explain a concept | Headline + 3-5 bullets | Medium | Feature breakdown |
| **Two-Column** | Compare or contrast | 50/50 or 60/40 split | Medium | Before/After, Pros/Cons |
| **Data/Chart** | Present metrics | Chart + headline + annotation | Medium-High | KPI dashboard |
| **Image + Text** | Illustrate concept | 60% image, 40% text | Low-Medium | Case study, Demo |
| **Quote** | Social proof, authority | Large quote, attribution | Minimal | Customer testimonial |
| **Process** | Show workflow/steps | Icons + labels, horizontal | Medium | How-it-works |
| **Team** | Introduce people | Photos + name + title | Low | About us |
| **CTA** | Drive action | Centered text + button/link | Minimal | Closing slide |

## Visual Hierarchy

| Level | Element | Font Size | Weight | Color |
|-------|---------|-----------|--------|-------|
| **1** | Slide headline | 28-36pt | Bold | Primary brand |
| **2** | Key data / number | 48-72pt | Bold | Accent color |
| **3** | Body / bullets | 18-24pt | Regular | Dark gray (#333) |
| **4** | Caption / source | 12-14pt | Light | Medium gray (#666) |

## Design Rules

| Rule | Specification | Why |
|------|-------------|-----|
| **Max bullets** | 5 per slide, 8 words each | Cognitive load limit |
| **Font pairing** | 1 sans-serif headline + 1 body font | Visual consistency |
| **Color palette** | Max 3 colors + neutrals | Reduces distraction |
| **Whitespace** | 20%+ of slide area empty | Improves readability |
| **Animations** | Fade-in only, no fly/spin/bounce | Professional appearance |
| **Charts** | Label directly on chart, no legends if possible | Faster comprehension |
| **Contrast** | 4.5:1 minimum (WCAG AA) | Accessibility |
| **Alignment** | Grid-based, consistent margins | Visual order |
