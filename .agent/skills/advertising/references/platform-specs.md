## Platform-Specific Compliance Checklist

**Google Search**:
- [ ] 15 headline variants created (3 active at minimum)
- [ ] 4 description variants available
- [ ] Final URL structure uses {keyword} insertion
- [ ] Ad group keywords at 8-12 terms (not <5, not >15)

**Meta (Facebook/Instagram)**:
- [ ] Primary text (first 125 chars) hooks within 3 words
- [ ] No hashtags in primary text, use on image only
- [ ] CTA button selected (Shop Now, Learn More, Sign Up, etc.)
- [ ] Image 1200×628px, legible text <20% image area

**LinkedIn**:
- [ ] Headline matches professional/B2B context (avoid urgency language)
- [ ] Intro text 150 chars, professional tone (no emojis)
- [ ] Image branded, vertical 1200×627px
- [ ] Audience: Title, company size, industry, seniority filters applied

**TikTok**:
- [ ] Caption authentic, trend-aware, under 150 chars
- [ ] No hard selling or corporate speak
- [ ] Video 9:16 aspect, 15-60 seconds, captions burned-in
- [ ] Sound + visual aligned to platform norms

**YouTube**:
- [ ] Title 100 chars, keyword-rich, hooks curiosity
- [ ] Description 30 chars, CTA explicit (link, button, subscribe)
- [ ] Thumbnail custom, text minimal, high contrast
- [ ] Pre-roll targeting: Intent keywords, remarketing lists, lookalike

## Disapproval Risk Assessment

**High Risk** (pause immediately):
- Unclear landing page relevance (>90% bounce rate)
- Missing brand name or value proposition in headline
- Duplicate ad copy across platforms without adaptation
- Prohibited categories (gambling, weapons, prescription meds without license)

**Medium Risk** (flag for review):
- CTR below platform baseline (-20% vs. category avg)
- Ad copy inconsistent with landing page (title mismatch)
- Competitor name in copy (defensive only, platform-specific rules apply)
