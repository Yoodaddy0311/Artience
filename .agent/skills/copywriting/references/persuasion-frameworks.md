# Persuasion Frameworks

**Core: Match framework to audience awareness level — AIDA for cold, PAS for pain-aware, BAB for solution-aware.**

## Framework Comparison

| Framework | Steps | Best For | Awareness Level |
|-----------|-------|----------|-----------------|
| **AIDA** | Attention → Interest → Desire → Action | Cold audiences, display ads, landing pages | Unaware → Solution-aware |
| **PAS** | Problem → Agitate → Solve | Pain-aware audiences, email, search ads | Problem-aware |
| **BAB** | Before → After → Bridge | Solution-aware, case studies, testimonials | Solution-aware → Most aware |
| **4Ps** | Promise → Picture → Proof → Push | Sales pages, product launches | All levels |
| **ACCA** | Awareness → Comprehension → Conviction → Action | Complex B2B, technical products | Unaware → Most aware |

## Headline Formula Library

| Pattern | Formula | Example | CTR Lift |
|---------|---------|---------|----------|
| **How-To** | How to [achieve X] without [pain Y] | How to Double Leads Without Increasing Ad Spend | +15-25% |
| **Number** | [N] [adjective] Ways to [benefit] | 7 Proven Ways to Reduce Cart Abandonment | +20-30% |
| **Question** | Are You [making mistake / missing out]? | Are You Losing 40% of Leads at Checkout? | +10-20% |
| **Proof** | How [entity] [achieved result] in [timeframe] | How Shopify Grew Revenue 3x in 6 Months | +25-35% |
| **Curiosity** | The [unexpected thing] That [surprising result] | The Email Trick That Doubled Our Open Rate | +15-25% |

## Power Words by Category

- **Urgency**: Now, Today, Limited, Instant, Deadline, Before, Hurry
- **Exclusivity**: Secret, Insider, Members-only, Invitation, Private
- **Trust**: Proven, Guaranteed, Certified, Research-backed, Tested
- **Value**: Free, Bonus, Save, Discount, Extra, Complimentary
- **Emotion**: Transform, Breakthrough, Unleash, Discover, Unlock

## Copywriting Checklist

- [ ] Framework matches audience awareness level
- [ ] Headline uses proven formula pattern
- [ ] Lead addresses reader's #1 pain point in first line
- [ ] Benefits stated before features (ratio 3:1)
- [ ] Social proof within first 30% of copy
- [ ] Single clear CTA per content piece
- [ ] Power words used in headline and CTA
- [ ] Read aloud — sounds conversational, not robotic
- [ ] Removed jargon; 8th-grade reading level max
- [ ] A/B test variant prepared for headline
