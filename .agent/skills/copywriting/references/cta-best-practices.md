# CTA Best Practices

**Core: Every CTA follows Action Verb + Benefit + Urgency. Weak CTAs describe; strong CTAs compel.**

## CTA Pattern Matrix

| Pattern | Weak Example | Strong Example | Use Case | Lift |
|---------|-------------|----------------|----------|------|
| **Action + Benefit** | Submit | Get My Free Guide | Lead gen forms | +30-40% |
| **Action + Urgency** | Sign Up | Start Free Trial — Today Only | SaaS signup | +20-30% |
| **Action + Specificity** | Learn More | See 3 Pricing Plans | Product pages | +15-25% |
| **First Person** | Start Your Trial | Start My Free Trial | Signup flows | +25-35% |
| **Loss Aversion** | Buy Now | Don't Miss 50% Off | E-commerce | +20-30% |
| **Social Proof** | Join | Join 10,000+ Marketers | Newsletters | +15-20% |
| **Low Commitment** | Purchase | Try Risk-Free for 14 Days | High-ticket items | +25-40% |

## CTA Button Design Rules

| Factor | Best Practice | Impact |
|--------|-------------|--------|
| **Color** | High contrast to page, not red (unless brand) | +10-20% CTR |
| **Size** | Min 44×44px touch target, prominent but not overwhelming | +5-10% |
| **Position** | Above fold primary, repeated after key sections | +15-25% |
| **Count** | 1 primary CTA per viewport, max 2 secondary | -20% if >3 |
| **Whitespace** | Min 20px padding around button | +10-15% |
| **Shape** | Rounded corners (4-8px radius) outperform sharp | +5-8% |

## Platform Copy Constraints

| Platform | Headline | Description | CTA Text | Image Text |
|----------|----------|-------------|----------|------------|
| **Google Search** | 30 chars ×3 | 90 chars ×2 | Auto | N/A |
| **Google Display** | 30 chars | 90 chars | Auto | <20% of image |
| **Meta (FB/IG)** | 40 chars | 125 chars primary | 25 chars | <20% of image |
| **LinkedIn** | 70 chars | 150 chars | Pre-set list | <20% of image |
| **Twitter/X** | 280 chars total | Incl. in tweet | Card: 20 chars | No limit |
| **Email** | 50 chars subject | Unlimited body | 25 chars button | Balance w/ text |

## Implementation Checklist

- [ ] CTA uses action verb (Get, Start, Try, Join, Download, Claim)
- [ ] Benefit is explicit, not implied
- [ ] Button color has 4.5:1+ contrast ratio
- [ ] One primary CTA above the fold
- [ ] CTA text under 5 words
- [ ] Mobile touch target ≥44×44px
- [ ] A/B test variant prepared (text or color)
- [ ] Urgency element added where appropriate
- [ ] First-person tested vs second-person
- [ ] Surrounding copy supports CTA with proof/benefit
