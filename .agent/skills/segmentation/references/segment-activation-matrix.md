# Segment Activation Matrix

**Core Principle: One Segment = One Coherent Experience**

After defining segments, activate them by tailoring channels, content, offers, and messaging. This matrix ensures consistency and measurability across all segment marketing.

## Segment Activation Strategy Matrix

| Segment | Primary Channel | Content Type | Offer Type | Tone | Frequency | Success Metric |
|---------|-----------------|--------------|-----------|------|-----------|----------------|
| **Enterprise Decision-Makers** | LinkedIn, email | Case studies, ROI calc, security brief | Custom pricing, SLA, support | Authoritative, strategic | 1-2x weekly | Sales pipeline value |
| **Mid-Market Buyers** | Email, webinar | Comparison guide, implementation guide, demo | Standard tier, discount, training | Professional, solution-focused | 2x weekly | Trial → paid conversion |
| **SMB Founders** | Social media, Product Hunt | Quick start guide, community stories, tips | Freemium, year prepay | Friendly, inspiring | 3x weekly | Freemium → paid |
| **Power Users** | In-app, email, community | Advanced guides, API docs, integration library | Pro tier, enterprise features | Technical, collaborative | 2x weekly | Feature adoption rate |
| **Casual Users** | Email, SMS | Tutorial videos, tips, success stories | Engagement offer, annual plan | Encouraging, helpful | 1x weekly | Monthly active users |
| **Churning Customers** | SMS, email, direct | Survey, special return offer, competitor alternative | 50% off 3 months, dedicated support | Apologetic, exclusive | Immediate | Reactivation rate |
| **Developer Personas** | GitHub, Stack Overflow, Slack | Code samples, API reference, integration guides | Early access, beta features | Technical, peer-to-peer | 1x weekly | GitHub stars, adopted integrations |

## Segment Activation Checklist

**Segment Definition** (before any marketing):
- [ ] 50+ customers/year in segment (or 5% of base)
- [ ] Demographics/behaviors clearly defined
- [ ] Persona developed with primary value prop
- [ ] Research: actual interview with 3+ in segment
- [ ] Success metric defined (specific, measurable)
- [ ] Owner assigned for segment

**Channel & Content**:
- [ ] Primary channel chosen with justification
- [ ] 3-5 content pieces developed for segment
- [ ] Messaging tested (clarity, relevance, credibility)
- [ ] CTA clear and specific to segment need
- [ ] Timing/frequency documented

**Measurement & Optimization**:
- [ ] Baseline metrics captured (email open rate, conversion, etc.)
- [ ] Cohort tracking enabled (segment in analytics)
- [ ] Success metric dashboard built
- [ ] Weekly review cadence scheduled
- [ ] Optimization triggers defined (e.g., <20% open rate = message test)

**Scale & Refine**:
- [ ] Segment proves repeatable (successful 2+ campaigns)
- [ ] Profitability > acquisition cost
- [ ] Process documented for future campaigns
- [ ] Team trained on segment persona and messaging
- [ ] Expansion to adjacent segment considered
