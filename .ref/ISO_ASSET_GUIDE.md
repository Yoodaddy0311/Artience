# 아이소메트릭 건물/가구 에셋 제작 가이드 (ISO Asset Guide)

AgentTown(오피스 맵)에 배치되는 건물이나 대형 가구 에셋을 제작할 때, 별도의 오프셋(Offset) 조정 없이 바닥 격자에 정확하게 맞물리도록 하기 위한 제작 규격입니다.

## 1. 아이소메트릭 그리드 규격 (Isometric Grid Scale)

AgentTown은 **2:1 비율의 아이소메트릭 투영법**을 사용합니다.
- **가로(Width) 비율**: 2
- **세로(Height) 비율**: 1
- **1칸(Tile)의 실제 픽셀 크기**: `64px (가로) x 32px (세로)`

## 2. 에셋 제작 시 캔버스 설정 (Photoshop / Illustrator)

에셋을 디자인할 때 **등각 투영(Isometric) 각도**를 정확히 맞춰야 선이 어긋나지 않습니다.

* **좌/우 벽면 기울기**: 정확히 `26.565도` (약 26.5도 또는 30도로 근사치 사용 금지)
* **포토샵/일러스트 팁 (2:1 비율 그리기)**:
  1. 가로 `2px`, 세로 `1px`씩 이동하면서 계단 현상으로 선을 긋습니다. (픽셀 아트 방식)
  2. 일반 도형(정사각형 등)을 그린 후, **수직 축소(Vertical Scale) 50%**, **기울기(Shear/Skew) 45도**, **회전(Rotate) -45도 (또는 +45도)** 변환을 순서대로 거치면 정확한 2:1 아이소메트릭 각도가 나옵니다.

## 3. 앵커 포인트 (Anchor Point) 맞추기

엔진에서 에셋을 렌더링할 때 투명한 캔버스의 특정 기준점(Anchor)을 격자 좌표(Grid Coordinate)와 일치시킵니다. 현재 엔진은 **하단 중앙 (0.5, 0.7)**을 기준으로 잡도록 설계되어 있습니다. 
-> `sprite.anchor.set(0.5, 0.7)`

따라서 에셋을 완성하고 WebP나 PNG로 내보낼 때 캔버스 여백을 다음과 같이 유지해주세요:

1. **에셋의 시각적 중심(바닥면의 중앙)**이 캔버스의 **가로 중앙(X 50%)**에 오도록 설정
2. **에셋의 바닥면(가장 낮은 꼭짓점)**이 캔버스의 **세로 하단 70% 지점(Y 70%)**에 오도록 여백 셋팅 (또는 에셋을 바닥 끝에 붙여놓고, 추후 Studio Mode에서 오프셋 튜닝 사용)

**가장 권장하는 방식:**
타일 크기(64x32)를 바닥에 베이스로 깔아둔 상태에서 건물을 올리고, 캔버스 사이즈를 타일에 딱 맞춘 박스(Bounding Box) 기준으로 크롭합니다. 그 후엔 Studio Mode의 방향키로 한두 번만 Nudge 해주면 완벽하게 맞물립니다.

## 4. 건물 크기 (Footprint)

기본적으로 건물 에셋은 **4x4** 또는 **6x6** 타일의 바닥 면적(Footprint)을 점유하도록 설계되어 있습니다.

- **6x6 타일 구조물 크기**: 가로 384px (6 * 64px), 세로 높이는 자유 (건물 층수 등에 따라 다름)
- 바닥 면과 평행하게 모서리를 그리면, 캐릭터가 건물을 타넘지 않고 자연스럽게 주위를 돌아다니게 됩니다.
